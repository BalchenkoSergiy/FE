1. Створити дирикторію `styles`, в середині директорії створити файл `main.css`
2. Підключити стилі до сторінки через тег `link`
3. Поміняти колір всього тексту на сторінці на `#555`
3. Поміняти колір заголовку `h1` на `darkblue`
4. Задати розмір фото по ширині в `200px`, та скруглити кути на `20%`
5. Для списку контактів задати колір фону `lightcyan`
6. Для списку технологій задати колір фону `lightsalmon`
